# 13 — Git Workflow

```
main
develop
feature/auth
feature/posts
feature/profile
feature/friends
feature/comments
```

**নিয়ম:**
- `main` → সবসময় production-ready
- `develop` → integration branch
- প্রতিটি feature নিজের branch-এ, develop-এ merge হয়ে তারপর main-এ যাবে
