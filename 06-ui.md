## Pages

| Page | Route |
|---|---|
| Home / Feed | `/` |
| Login | `/login` |
| Register | `/register` |
| Profile | `/@[username]` |
| Friends | `/friends` |
| Messages | `/messages` *(V2)* |
| Notifications | `/notifications` *(V2)* |
| Settings | `/settings` |
| Search | `/search` |
